# Sales Data Cleaning & Analysis (Excel Project)

## Project Overview
This project focuses on cleaning and analyzing messy retail sales data using Microsoft Excel.

## Tasks Performed
- Removed duplicates
- Standardized date formats
- Cleaned text inconsistencies
- Handled missing and invalid values
- Recalculated total sales
- Created pivot table analysis
- Created pivot charts visualization
- Created a dashboard with filters

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel formulas
- Dashboard


## Key Insights
- Identified top-performing products
- Analyzed sales by city and category
- Determined most used payment methods


##Outcomes
- Cleaned and standardized a retail dataset of 20+ customer orders in Excel by removing duplicates, fixing date formats, correcting text inconsistencies, and handling missing/invalid values.
- Built 3 pivot tables and charts to perform descriptive analytics, identifying top‑performing products, city/category sales trends, and customer payment preferences.
- Delivered actionable insights by transforming raw sales data (₱224K revenue across 3 cities) into a structured dashboard, improving reporting clarity by 100% (messy → fully standardized) and showcasing analytical skills through 3 interactive visualizations

Descriptive Analytics
